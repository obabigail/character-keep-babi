# MPV Character Keep Babi

*Character Keep Babi* é um site em Streamlit que permite que auxilia o usuário a **criar, editar e exportar fichas de RPG de mesa**

O MPV do sistema deve contar com:

- Pelo menos um sistema de RPG conhecido como opção de ficha
- Uma interface amigável com passos divididos em tabs
- Uma funcionalidade para baixar as fichas criadas em PDF (modelo oficial);
- Uma forma de amazenar, carregar e editar fichas criadas
